# 🎓 Students Performance Analysis (EDA)

A complete **Exploratory Data Analysis (EDA)** project on the **Students Performance Dataset** using **Python, Pandas, Matplotlib, and Seaborn**. This project explores the factors that influence students' academic performance and uncovers meaningful insights through data visualization and statistical analysis.

---

## 📌 Project Overview

Education data contains valuable information that can help identify the factors affecting student performance.

In this project, we analyze students' scores in **Math, Reading, and Writing**, create new performance metrics, visualize trends, and discover which factors contribute most to academic success.

The notebook covers everything from data exploration to feature engineering and insightful visualizations.

---

## 🎯 Objectives

- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Create new performance-related features
- Explore score distributions
- Analyze relationships between categorical and numerical variables
- Discover patterns affecting student performance
- Generate actionable educational insights

## 🔍 Key Insights

Some important observations include:

- Students who completed the **Test Preparation Course** consistently scored higher.
- Reading and Writing scores show a strong positive correlation.
- Students with higher average scores fall mostly into the **Excellent** and **Good** categories.
- Lunch type has a noticeable relationship with academic performance.
- Parental education level shows a positive impact on student scores.
